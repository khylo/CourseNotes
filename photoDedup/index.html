Scan your Google Photos library to identify exact duplicates, burst shots, and similar images — then delete with confidence using side-by-side previews.

Setup (one time)

1

Go to Google Photos API console, create a project, enable the Photos Library API, and create an OAuth 2.0 credential (type: Web app).

2

Add http://localhost:8080 as an authorised redirect URI in your OAuth credentials.

3

Serve this file locally: python3 -m http.server 8080 — then open http://localhost:8080

4

Enter your OAuth Client ID below and click Connect.

— or —

This app runs entirely in your browser. No data leaves your machine.
